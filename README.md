Vault Management\n```rust\npub fn initialize_vault(ctx: Context<InitializeVault>) -> Result<()>\n```\n- Creates a **PDA-based vault** (one canonical vault per program)\n- Sets admin and initializes totals\n- Emits `VaultInitialized` event\n\n### 2️⃣ Deposit with Tracking\n```rust\npub fn deposit(ctx: Context<Deposit>, amount: u64) -> Result<()>\n```\n- **Transfers SOL** from user to vault\n- **Tracks per-user totals** (amount + deposit count)\n- **Auto-counts new users**\n- **Overflow protection** with checked arithmetic\n- Emits `Deposited` event with updated stats\n\n### 3️⃣ Query Functions (View)\n```rust\npub fn get_user_balance(ctx: Context<GetUserBalance>) -> Result<u64>\npub fn get_vault_stats(ctx: Context<GetVaultStats>) -> Result<VaultStats>\n```\n- No-cost queries for UI updates\n- Returns: `total_deposited`, `total_users`, `total_deposits`, `average_deposit`\n\n### 4️⃣ Admin Controls\n```rust\npub fn withdraw(ctx: Context<Withdraw>, amount: u64) -> Result<()>\npub fn toggle_pause(ctx: Context<TogglePause>) -> Result<()>\n```\n- Admin-only withdrawals\n- Pause/resume deposits\n- Signer verification on all operations\n\n## 🎨 Live Dashboard\n\n### Components\n-
